ensureScriptSettingsDefaults

fun ensureScriptSettingsDefaults(packConfig: PackConfig)(source)

Update the script settings and ensure the default keys, with values gathered from the passed PackConfig, are present:

  1. SPC_SERVERPACKCREATOR_VERSION_SPC : ServerPackCreator version with which the scripts were created

  2. SPC_MINECRAFT_VERSION_SPC : Minecraft version of the modpack ` *

  3. SPC_MODLOADER_SPC : The modloader of the modpack

  4. SPC_MODLOADER_VERSION_SPC : The modloader version of the modpack *

  5. SPC_JAVA_ARGS_SPC : The JVM args to be used to run the server

  6. SPC_JAVA_SPC : Path to the java installation to be used to run the server

  7. SPC_FABRIC_INSTALLER_VERSION_SPC : Most recent version of the Fabric installer at the time of creating the scripts

  8. SPC_QUILT_INSTALLER_VERSION_SPC : Most recent version of the Quilt installer at the time of creating the scripts
